# Agente Inteligente de Jogo da Velha (A3)

Este projeto implementa um agente de Inteligência Artificial capaz de jogar Jogo da Velha utilizando uma arquitetura baseada em Regras de Decisão e Orientação a Objetos.

## Requisitos do Sistema
* Python 3.x instalado.
* Nenhuma biblioteca externa é necessária (utiliza apenas bibliotecas padrão: `random`').

## Como Executar (Script de Inicialização)

1. Certifique-se de estar na pasta `codigo fonte`.
2. Abra o terminal ou prompt de comando nesta pasta.
3. Execute o comando abaixo para iniciar o sistema:

python jogo_da_velha.py

## Funcionalidades
* **Modo Humano vs IA:** Permite desafiar o agente inteligente.

## Arquitetura
O sistema é dividido em:
* **Ambiente:** Gerencia o tabuleiro e regras.
* **AgenteIA:** Implementa a lógica de decisão (Ataque, Bloqueio, Estratégia).
